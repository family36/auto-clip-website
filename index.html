プライバシーポリシー

AutoClip（以下「当アプリ」）は、ユーザーが許可した範囲で TikTok API にアクセスし、動画のアップロードを自動化します。

  • 取得データ：TikTok アカウント ID、アクセストークン、有効期限
  • 利用目的：動画アップロード機能の提供
  • 第三者提供：取得したデータを第三者に提供することはありません
  • 保存期間：ユーザーが連携を解除するか、1 年間利用がない場合にデータを削除します
  • お問い合わせ：autocliptiktok@gmail.com

利用規約

当アプリを利用することで、ユーザーは TikTok の利用規約および本規約に同意したものとみなされます。

  • 本アプリはベータ版であり、動作保証は行いません
  • ユーザーがアップロードするコンテンツの権利・責任はユーザーに帰属します
  • 予告なくサービスを停止・変更する場合があります
